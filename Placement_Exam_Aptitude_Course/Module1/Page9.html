Algebra

Concept Overview

Algebra involves working with symbols and rules for manipulating these symbols. It forms the core of mathematical reasoning.

1. Linear Equations

A linear equation is an equation of the first degree. It can have one or more variables.

2. Quadratic Equations

A quadratic equation is of the form ax² + bx + c = 0.

Roots: x = [-b ± √(b² - 4ac)] / 2a

3. Polynomials and Identities

  • (a + b)² = a² + 2ab + b²
  • (a - b)² = a² - 2ab + b²
  • a³ + b³ = (a + b)(a² - ab + b²)

4. Inequalities

Inequalities express a relationship of comparison between two values. (>, <, ≥, ≤).
