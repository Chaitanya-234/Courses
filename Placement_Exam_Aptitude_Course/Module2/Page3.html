Coding and Decoding

Concept Overview

Coding and Decoding involves deciphering a pattern or rule used to encode a message and applying that rule to decode another message.

1. Letter Coding

Letters are replaced by other letters according to a pattern (+1, -1, Reverse).

Example: CAT -> DBU (+1).

2. Number Coding

Letters are assigned numerical values (A=1, B=2).

Example: BAD = 2+1+4 = 7.

3. Substitution Coding

Words are substituted with other words.

Example: 'White' is called 'Blue'.

Detailed Explanation: Mixed Coding

Comparing sentences to find codes for specific words.
