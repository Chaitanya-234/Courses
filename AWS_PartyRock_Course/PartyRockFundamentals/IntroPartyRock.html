Introduction to PartyRock

What is AWS PartyRock?

PartyRock is a free, no-code playground powered by Amazon Bedrock that lets anyone build and share GenAI applications. It requires no AWS account and is designed for rapid prototyping and learning.

Key Features

  • No AWS account needed: Sign in with social accounts to start building
  • Widget-based builder: Drag and drop text, image, and chatbot widgets
  • Multiple models: Access Claude, Titan, Llama, and other Bedrock models
  • Instant sharing: Generate shareable links for your apps
  • Free tier: Generous free usage for learning and prototyping

Building Your First PartyRock App

  1. Go to partyrock.aws and sign in
  2. Click "Build your own app" or describe what you want in natural language
  3. PartyRock generates an initial app layout with widgets
  4. Customize widget prompts, model selection, and layout
  5. Test and iterate on your design

Prompt Engineering in PartyRock

Each widget has a prompt that can reference other widgets using @WidgetName:

Widget: "Story Generator"
Model: Claude 3 Sonnet
Prompt: "Write a short story about @Topic in the style of @Genre.
The story should be @Length paragraphs long and include the character @CharacterName."
